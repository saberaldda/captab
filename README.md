## What it does

CapTab is the operating system pharmacies use to run their day-to-day operations — inventory, sales, customers, suppliers, analytics — across a Filament-powered web admin panel and offline-first mobile apps for iOS and Android. The mobile clients keep working at the counter whether the connection is up or not, then sync automatically. The whole product ships bilingual (Arabic / English) from day one.

## Features

- **Smart inventory** — stock levels, expiration tracking, batch movements, stocktaking workflows
- **Point of sale** — fast invoicing, payments, customer credit, daily settlements
- **Customer profiles** — preferences, allergies, prescription history, loyalty
- **Supplier management** — order requests, deliveries, supplier transactions
- **Live analytics** — sales, profits, expenses, employee performance
- **Multi-tenant architecture** — one platform serving multiple independent pharmacies with isolated data, roles, and permissions
- **Bilingual (Arabic / English)** — full RTL support across web admin and mobile apps
- **Offline-first mobile** — local-first POS with background sync when the connection returns

## Platforms

- **Web admin panel** — Laravel 11 + Filament 3, used by pharmacy managers and CapTab operators
- **Mobile apps** — iOS and Android, used by pharmacy staff at the counter
- **REST API** — Laravel Sanctum + JWT, powers both mobile apps and partner integrations

## Tech stack

**Backend** — PHP 8.2, Laravel 11, MySQL, Filament 3, Spatie Permissions, Spatie Translatable, Laravel Sanctum, JWT, Laravel Telescope
**Frontend** — Tailwind CSS, Vite, Livewire (admin); native mobile clients
**Infra** — Docker, queue workers, scheduled jobs, Excel import/export pipelines
